FAQ

What OS is supported?

Designed and tested on Windows 11. As a standard USB‑HID joystick, the core device also works on other OSes that support HID, but the companion tuner app is currently Windows‑only.

Which controllers are supported?

Asciiware Sphere 360, SpaceOrb 360, SpaceBall 4000/5000/3003, and Magellan Space Mouse Plus—plus additional RS‑232 NDOF devices via profiles.

Is any soldering required?

No for the finished adapter. For DIY builds, you’ll need to follow the wiring guide included with the firmware repository.

Does it work with CAD and games?

Yes—VectorBridge presents as a standard joystick, so anything that can read HID axes/buttons can use it. App‑specific profiles in the tuner help dial things in.
