VectorBridge Software

Two included Windows apps show the real value of VectorBridge: the Tuner that writes your settings to the adapter’s NVM, and the Mouse&Keyboard Feeder that makes classic 6‑DoF controllers feel at home in modern games.

Tuner — Device Selection

Tuner highlights

  • Choose from supported devices (e.g., SpaceOrb 360, SpaceBall series, Magellan Classic/Plus)
  • Adjust sensitivity, dead‑zones, and response curves
  • Writes to device NVM — settings travel with your VectorBridge
  • Starter profiles let you begin quickly; refine with sliders
  • Windows 10/11
Screenshot guidance
  1. tuner_device_selection.png — single window, dropdown visible with SpaceOrb 360 selected; keep other UI minimal.
  2. tuner_sliders.png — show sliders for sensitivity/dead‑zone/curves (no device dropdown needed).
  3. tuner_write_to_device.png — show the control/status for writing to NVM.
  4. tuner_starter_profiles.png — show the starter profiles list.
Mouse&Keyboard Feeder — Launch and Play

Feeder highlights

  • Play virtually any FPS — maps to standard mouse/keyboard conventions
  • Map axes and buttons to any key or mouse function
  • Save / Load / Import / Export profiles; easy sharing
  • Windows 10/11
Screenshot guidance
  1. feeder_main.png — main window showing mappings overview.
  2. feeder_mapping.png — detailed mapping editor (axis/button → key/mouse).
  3. feeder_profiles.png — profiles area demonstrating import/export.
  4. feeder_mouselook.png — show mouse‑look related settings for FPS.
